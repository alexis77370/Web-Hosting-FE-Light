 - Options
database -> Nom de la base de donnees et de sont utilisateur
password -> Mot de passe de l'utilisateur de la base de donnees

 - Ajouter une base de donnees
webhostingfe.sh mysql create "database" "password"

 - Modifier le mot de passe d'une base de donnees
webhostingfe.sh mysql set "database" "password"

 - Supprimer une base de donnees
webhostingfe.sh mysql delete "database"
